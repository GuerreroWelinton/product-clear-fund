# Catálogo inicial de errores

| Código | Significado |
|---|---|
| AUTH_UNAUTHENTICATED | No existe sesión válida |
| AUTH_FORBIDDEN_CASH_FUND | El usuario no puede acceder a la caja |
| CASH_FUND_INACTIVE | La caja está inactiva |
| CASH_FUND_FIXED_FEE_LOCKED | La cuota mensual ya no puede modificarse |
| PERSON_DOCUMENT_EXISTS | La cédula ya pertenece a otra persona |
| MEMBERSHIP_EQUALIZATION_INCOMPLETE | La igualación no está pagada completamente |
| SAVINGS_OLDER_OBLIGATION_PENDING | Existen cuotas anteriores pendientes |
| PAYMENT_FUTURE_DATE | La fecha del pago es futura |
| PAYMENT_ALREADY_ALLOCATED | La obligación ya fue pagada |
| LOAN_TEMPLATE_INVALID_CAPITAL | Los tramos no suman el principal |
| LOAN_PERSON_ALREADY_HAS_ACTIVE | La persona ya tiene préstamo bloqueante |
| LOAN_SAVINGS_NOT_CURRENT | Los ahorros requeridos no están al día |
| LOAN_INSUFFICIENT_FREE_BALANCE | No existe saldo libre suficiente |
| LOAN_TURN_NOT_AVAILABLE | El turno ya no está disponible |
| WITHDRAWAL_NEGATIVE_SETTLEMENT | El retiro deja una diferencia pendiente |
| CLOSING_NOT_DISTRIBUTABLE | No existe rendimiento positivo para repartir |
